# Employee Management System 💼

A **Database Management System (DBMS)** project developed to efficiently manage employee information, attendance, salary, performance, departments, training, and projects.

## 📌 Project Overview

The Employee Management System is designed to organize and manage employee-related data in a structured way using **MySQL**. It helps in maintaining employee records, attendance, salary details, training programs, project allocation, and performance tracking.

## ✨ Features

* Employee Information Management
* Attendance Tracking
* Salary & Leave Management
* Performance Monitoring
* Department & Branch Management
* Employee Training Records
* Project Management
* Efficient Database Relationships using ER Model

## 🛠 Technologies Used

* MySQL
* DBMS Concepts
* SQL Queries
* ER Model

## 📂 Database Modules

* Employee Table
* Employer Table
* Job Role Table
* Department Table
* Branch Table
* Training Table
* Attendance Table
* Salary Table
* Leave Table
* Performance Table
* Project Table

## 🎯 Project Objective

The main objective of this project is to provide an efficient and organized system for storing and managing employee-related information while maintaining proper database relationships.
